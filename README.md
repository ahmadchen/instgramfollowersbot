# Instagramfollowersbot

Best Tool For Increase Instagram Follower.

## Requirements
1. openssl
2. curl

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/ahmadchen/instgramfollowersbot`

`$ cd instgramfollowersbot`

`$ chmod +x run.sh`

`$ termux-wake-lock`

`$ bash run.sh`
